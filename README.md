<details>
<summary><b>🍵 Matcha Scout — AI-Powered Matcha Discovery Platform</b></summary>
<br>

> An end-to-end production application that parses free-text reviews into structured taste profiles and ranks cafe drinks against a user's preferences using explainable, deterministic recommendation scoring.

| Attribute | Details |
|:---|:---|
| **Stack** | Next.js 16 · React 19 · TypeScript · Tailwind CSS · FastAPI · Python · Pydantic · DynamoDB · AWS Lambda · API Gateway · Vercel |
| **Scale** | Multi-region production deployment — San Diego + Orange County |
| **Performance** | Fully serverless AWS backend; Lambda + API Gateway + DynamoDB; zero always-on infrastructure cost |
| **Security** | Environment-gated AI mode; mock parser default for safe operation; no Yelp data stored as user reviews |
| **DevOps** | Docker Compose local dev · verified kind Kubernetes manifests · AWS SAM IaC deployment |
| **Impact** | Live at [matcha-scout.vercel.app](https://matcha-scout.vercel.app) · 500+ Vercel deployments · multi-city Yelp Fusion API ingestion |
| **Repository** | [github.com/hhavynn/matcha-scout](https://github.com/hhavynn/matcha-scout) |

- Gemini-compatible structured review parsing pipeline mapping natural language → 5 taste dimensions
- Confidence scoring system: Unrated / Low / Medium / High based exclusively on community review count
- Multi-city Yelp Fusion API ingestion with deduplication and admin-controlled manual drink curation workflow
- DynamoDB data modeling across cafe / drink / review / taste-profile entities with local and production paths
- Preference quiz across 5 taste dimensions with explainable ranked match results and per-match reasoning
